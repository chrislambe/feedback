What is Ruby?

  • Object-oriented
  • Dynamic/duck typing
  • Package/library management via RubyGems
  • Many implementations including JIT and pre-compiling
  • Focused on programmer productivity and enjoyment: ...we need to focus on humans, on how humans care about doing programming or operating the application of the machines. We are the masters. [Machines] are the slaves.

Application architecture (it ain't gonna plan itself)

  • User
    • name:string
    • email:string
  • Submission
    • user:references
    • body:text

Lay the groundwork

  1. gem install rails
  2. rails new .
  3. Add rails-haml gem to our Gemfile and run: bundle install

Generate resources

  1. rails g scaffold User name:string email:string
  2. rails g scaffold Submission user:references body:text
  3. rake db:migrate

Fix funky output in the generated code

  1. Change the User reference to a dropdown list in the Submission form
  2. Change User#index and User#show to display User.name instead of just User

Make it usable

  1. Add navigation
  2. Add per-user submission lists

Tell us when we get feedback

  1. Tell ActionMailer to use sendmail
  2. rails g mailer SubmissionMailer
  3. Create submission notification method
  4. Create submission notification view
  5. Implement submission notification on SubmissionController#create

Deploy it
